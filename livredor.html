⚙️ Configuration requise

Pour activer le livre d'or, suivez ces étapes :

  1. Créez un compte gratuit sur supabase.com
  2. Créez un nouveau projet
  3. Dans l'éditeur SQL, exécutez le script fourni ci-dessous
  4. Copiez votre URL et votre anon key (Settings → API)
  5. Remplacez VOTRE_URL_SUPABASE et VOTRE_ANON_KEY en bas de cette page
  6. Choisissez un mot de passe admin et remplacez MOT_DE_PASSE_ADMIN

Script SQL à exécuter dans Supabase : 
create table messages (
  id uuid primary key default gen_random_uuid(),
  nom text not null,
  texte text not null,
  etoiles int default 5,
  reponse text,
  created_at timestamptz default now()
);
alter table messages enable row level security;
create policy "lecture publique" on messages for select using (true);
create policy "insertion publique" on messages for insert with check (true);
create policy "mise a jour admin" on messages for update using (true);
create policy "suppression admin" on messages for delete using (true);
